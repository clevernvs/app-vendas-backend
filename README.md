<h1> Descrição: </h1>

Backend da aplicação de vendas.

<h1> Instalando das dependências: </h1>

```bash
$ npm install
```

<h1> Utilizando a aplicação: </h1>

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

<h1> Testes: </h1>

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
